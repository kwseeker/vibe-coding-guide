# UIUX 设计提示词

**提取网站提示词**：

- [**DesignPrompts模版**](https://www.designprompts.dev/)
	提供了30个模版网页，可以直接获取网页设计风格提示词。
- **使用 `ui-ux-pro-maxs` 技能**
	此技能中同样提供了几十套模版网页设计风格提示词。
- **网页到Prompt的转换工具**
	（其实没有多大技术含量）
	- Chrome SuperDesign 插件（克隆免费、转提示词收费，按月收费且有点贵）
		通过 Chrome SuperDesign 插件克隆网页的样式，然后让AI转成类似 ui-ux-pro-max 那种规范的样式提示词（插件虽然也可以直接转不过要收费）。
	- image2prompt （也是要收费的，不过一次性买断）
		对网页进行截图，上传截图转成设计提示词。
