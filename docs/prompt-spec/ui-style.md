# UI 风格提示词

UI风格提示词需要描述下面内容：

- 概览（Overview）
- 配色方案（Color Palette）
- 字体排版（Typography）（重点关注字体粗细、字号，以及项目中不同字体是如何搭配和使用的）
- 间距系统（Spacing System）
- 组件样式（Component Styles）
- 阴影与层级（Shadows & Elevation）
- 动画与过渡（Animations & Transitions）
- 圆角（Border Radius）
- 不透明度与透明度（Opacity & Transparency）
- 项目中 Tailwind CSS 的通用用法（Common Tailwind CSS Usage in Project）
- 示例组件参考设计代码（Example component reference design code）
- 等等……

可以参考 ui-ux-pro-max MASTER.md 其本身也是一个 UI 风格指南。

每做完一个项目，可以让AI帮忙总结 UI 风格指南，后面可以复用到其他项目中。
